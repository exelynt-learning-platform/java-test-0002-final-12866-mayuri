# Mirrored Number Pyramid – Java

## Description

This program prints a mirrored number pyramid using nested loops in Java.

The pattern generated by the program is:

```
    1
  1 2 1
1 2 3 2 1
```

1 2 3 4 3 2 1
1 2 3 4 5 4 3 2 1

## Concepts Used

* Nested loops
* Number sequencing
* Pattern formatting
* Console output in Java

## How to Run

Compile the program:

```
javac NumberPattern.java
```

Run the program:

```
java NumberPattern
```
